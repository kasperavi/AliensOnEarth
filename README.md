# AliensOnEarth
Aliens On Earth Multunus Programming </br>

*Download the Project and open in Visual Studio 2012 + </br>
*Add the ITextSharp reference to the project and the project can now run.</br>
*Adding a new format is easy and you dont have to edit the application code.</br>
*To create a format extend your class to the Interface named IGenerateFormat</br>
*Enter the Format during user input correctly to execute the classes using Reflections
